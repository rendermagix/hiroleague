## 1. Install the server

The server runs on your desktop (Root Node) and is managed through `phbcli`.

Install [uv](https://docs.astral.sh/uv/getting-started/installation/), then install `phbcli`:

```bash
uv tool install phbcli
```

Initialize a workspace and start the server:

```bash
phbcli setup --workspace home
```

See [Setup and uninstall](/phb/setup-and-uninstall) for all setup options.

## 2. Install the gateway

The gateway handles authentication and routes traffic between your devices and the server.

Install [uv](https://docs.astral.sh/uv/getting-started/installation/), then install `phbgateway`:

```bash
uv tool install phbgateway
```

Create and start a gateway instance:

```bash
phbgateway instance create home --host 0.0.0.0 --port 8765 --desktop-pubkey "<base64-public-key>" --set-default
phbgateway start
```

See [Gateway CLI commands](/phb/gateway-cli) for the full setup flow.

## 3. Install the mobile or web app

Install [Flutter](https://docs.flutter.dev/install/quick), then build and run the app:

```bash
flutter run
```

Once the app is running, pair your device through the app to connect it to your workspace.
