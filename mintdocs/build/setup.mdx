## PHBServer / phbcli

Install [uv](https://docs.astral.sh/uv/getting-started/installation/) — the Python package and project manager used by the server.

## Mobile / web app

Install [Flutter](https://docs.flutter.dev/install/quick).

## Documentation (optional)

Install the [Mintlify CLI](https://www.mintlify.com/docs/quickstart) to preview the docs site locally and to set up AI tools in Cursor.

```bash
npm i -g mint
```

## Diagram generator

Follow the instructions in [`diagram-gen/README.md`](https://github.com/privatehomebox/privatehomebox/blob/main/diagram-gen/README.md) to install the Mermaid diagram generator.
