# Architecture Philosophy

This is a Desktop First Project. This is where the user trusts to have their own data, control, and privacy.

# Communication

There are a few options for external communication:
- Telegram
- Whatsapp
- Tailscale
- VPN
- Relay Gateway

  - WS/Json-RPC for communication protocols

Whatsapp requires a new Phone Number - maybe eSim. 
Telegram requires setting up a bot.

Markdown Table:

| Method     | Requirements | UI | Messages | Channels | 
|------------|-------------|----------|----------|----------|
| Telegram   | setup bot   | Requires a bot |
| Whatsapp   | New Phone Number | Requires a new phone number |
| Tailscale  | Free Account | Requires a new phone number |
| VPN        | Easy to set up | Requires a new phone number |
| Relay         | Easy to set up | Requires a new phone number |
